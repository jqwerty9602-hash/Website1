# Task: Recreate DHL Homepage Layout (Localhost Version)

## Objective
Recreate the visual layout and user experience inspired by
https://www.dhl.com/my-en/home.html
using **100% original HTML and CSS**, runnable on **localhost**.

⚠️ STRICT RULES
- DO NOT copy or scrape any DHL source code
- DO NOT reuse DHL class names, IDs, or assets
- DO NOT use DHL logos, icons, or copyrighted images
- Output must be original and legally safe

## Environment
- Localhost only
- No build tools
- No bundlers
- No server-side rendering
- Files must open via:
  http://localhost/ OR http://127.0.0.1/

## File Structure
/project-root
 ├── index.html
 ├── css/
 │   └── style.css
 └── assets/
     ├── images/
     └── icons/

## Layout Sections to Recreate (Visual Only)
1. Top navigation bar
2. Hero section with background image + CTA
3. Shipment tracking form (UI only)
4. Service cards / features grid
5. Promotional content section
6. Trust / stats section
7. Footer with multi-column links

## Design Requirements
- Desktop-first, fully responsive
- Use Flexbox and CSS Grid only
- Clean corporate look
- Color inspiration similar to logistics brand
  (yellow/red inspired but NOT DHL exact colors)
- Use system fonts (no Google Fonts CDN)

## Image Rules
- Use local placeholder images
- Use gradient backgrounds if images are missing
- No external image URLs

## HTML Requirements
- Semantic HTML5
- Accessible structure (nav, main, section, footer)
- Clean class naming (BEM or readable custom style)
- Comment each major section

## CSS Requirements
- One CSS file only
- Mobile responsive via media queries
- No CSS frameworks
- No inline styles

## Output Rules
- Provide ONLY:
  1. index.html
  2. css/style.css
- No explanations
- No markdown formatting
- Code only
